# TokPro (TikTok Pro MVP)

## Install

```bash
npm install
```

## Run

```bash
npx expo start
```

From the Expo CLI:
- Press `i` for iOS simulator (macOS only).
- Press `a` for Android emulator.
- Or scan the QR code with the Expo Go app on your phone.

## Features
- Full-screen vertical video feed (For You).
- Save flow with folder picker and folder creation.
- Saved folders and folder detail views.
- Persistent storage via AsyncStorage.

## Notes
- Videos are streamed from public MP4 sample URLs.
- Tabs match TikTok-style layout (Home, Friends, Create, Inbox, Profile).
